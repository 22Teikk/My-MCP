# My-MCP

## Figma MCP
- Reduces design–code discrepancies
- Speeds up UI development
- Maintains brand and UX consistency

## Gitlab Knowledge Graph (GKG)
- Connects and visualizes relationships
- Helps developers understand how features, bugs, and commits are linked across the project lifecycle.
- Improve AI handle difficult task with index code base

## MarkItDown
- MCP convert Document(docx, pdf, audio, zip, images, link,...) to .md
- Using CLI or API in Python
- Support large type document files

## Mobile MCP
- MCP can control mobile device end-to-end
- No need to write automation script 

## Context7
- Ensures up-to-date APIs
- Faster SDK integration
